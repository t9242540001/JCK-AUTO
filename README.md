# JCK AUTO Website

Корпоративный сайт компании JCK AUTO — импорт автомобилей из Китая, Кореи и Японии.

## Технологии

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Framer Motion
- MDX (блог)

## Запуск

```bash
cd jck-auto
npm install
npm run dev
```

Откройте http://localhost:3000

## Деплой

Проект настроен для деплоя на Vercel. При подключении репозитория укажите **Root Directory: `jck-auto`**

## Структура

- `/` — Главная страница (9 секций)
- `/calculator` — Калькулятор стоимости импорта
- `/about` — О компании и команде
- `/blog` — Статьи об импорте авто
