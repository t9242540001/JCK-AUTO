# ТЗ: Каталог ноускатов — контекст и архитектура

**Версия:** 1.0 | **Дата:** 2026-04-08 | **Ветка:** `claude/news-pipeline`

---

## Суть задачи

Создать раздел `/catalog/noscut` — каталог комплектов для восстановления передней части автомобиля (ноускат) для самых популярных в России импортных автомобилей. Каталог генерируется и обновляется автоматически через AI-pipeline. Одновременно: универсальная форма захвата лидов, рефакторинг URL каталога.

---

## Бизнес-логика

**Ноускат** — комплект деталей передней части: бампер, оптика, радиатор, телевизор, датчики, камера. Поставляется из Азии под заказ ~30 дней. Цена комплекта в 2–3 раза ниже аналогичного набора деталей на российском рынке (ориентир: от 199 000 ₽ против 350–600 000 ₽ поштучно).

**Аудитория:** кузовные СТО (основной B2B канал) и частные владельцы с повреждённым передом (B2C). Оптовые условия для СТО — отдельная CTA на каждой карточке.

**Ценовая стратегия:** единая стартовая цена `от 199 000 ₽` как маркетинговый якорь. Если автоматически найдена среднерыночная цена — показываем сравнение и экономию в процентах.

**Доставка:** по всей России и СНГ. Автоматически сгенерированный список из 5 городов России + 3 столицы СНГ с ориентировочными ценами и сроками. Генерирует DeepSeek. Дисклеймер обязателен. Пометка «Отправим куда скажете».

---

## Фильтр возраста кузовов

Берём кузов если `yearEnd >= currentYear - 5`, где `currentYear` — текущий год на момент запуска скрипта (динамическое значение).

В 2026 году: `yearEnd >= 2021`. В 2027: `yearEnd >= 2022`. Список автоматически «стареет» вместе с рынком.

Кузов BMW X5 F15 (2013–2018): `2018 >= 2021` — false, не берём.
Кузов BMW X5 G05 (2018–2025): `2025 >= 2021` — true, берём.

## Структура URL
```
/catalog                     → хаб (авто в наличии + категории запчастей)
/catalog/cars/[id]           → карточка авто (перенесено из /catalog/[id])
/catalog/[id]                → 301 redirect → /catalog/cars/[id]
/catalog/noscut              → список ноускатов (~100 карточек)
/catalog/noscut/[slug]       → карточка ноуската
```

Навигационный пункт «Каталог» в меню остаётся на `/catalog`. Подменю добавляем при появлении третьей категории.

## Структура данных

### `/storage/noscut/models.json`
```json
[{
  "make": "Toyota", "model": "RAV4", "generation": "XA50",
  "yearStart": 2019, "yearEnd": 2024,
  "slug": "toyota-rav4-xa50", "country": "japan"
}]
```

### `/storage/noscut/noscut-catalog.json`
```json
[{
  "slug": "toyota-rav4-xa50", "make": "Toyota", "model": "RAV4",
  "generation": "XA50", "yearStart": 2019, "yearEnd": 2024,
  "country": "japan", "priceFrom": 199000, "inStock": false,
  "components": ["бампер", "оптика", "радиатор", "телевизор", "датчики", "камера"],
  "marketPriceRu": null, "marketPriceSource": null, "marketPriceUpdated": null,
  "image": "/storage/noscut/toyota-rav4-xa50.jpg",
  "description": "...", "updatedAt": "2026-04-08"
}]
```

### `/storage/noscut/noscut-instock.json`
Whitelist моделей «в наличии», редактируется вручную:
```json
["toyota-camry-v70", "hyundai-solaris-iii", "kia-rio-iv"]
```

### `src/lib/deliveryConfig.ts`
Генерируется скриптом `research-noscut-models.ts` через DeepSeek. Содержит 5 городов России + 3 столицы СНГ с `priceFrom` и диапазоном дней. Дополняется пометкой «Отправим в любой город». Не редактируется вручную.

## Компоненты изображений

Генерируются через Qwen-Image-2.0-Pro (DashScope). Стиль — exploded view:
- Светло-серый нейтральный фон
- Силуэт **конкретной модели и поколения** в 3/4 ракурсе
- Строго 6 компонентов с пронумерованными выносками: бампер, оптика, радиатор, телевизор, датчики, камера
- Остальные части кузова затемнены/прозрачны
- **Изменение состава компонентов — только с явного согласия владельца**

Изображение на карточке — аккуратного размера, **без lightbox**. Акцент на названии модели, тексте и форме захвата.

## Универсальная форма захвата (LeadForm)

Новый компонент `src/components/LeadForm.tsx`. Встраиваемый (не модальный).
Пропсы: `subject` (тема заявки), `ctaLabel` (текст кнопки, дефолт «Оставить заявку»), `compact` (boolean).

Поля: телефон (обязательный) + имя (необязательное) + комментарий (необязательный).
Три состояния: idle / loading / success. При success: «Менеджер свяжется в ближайшее время».

Вторичная ссылка под кнопкой: «Вы представляете СТО? Условия для оптовых покупателей →» (Telegram `jck_auto_manager`).

Существующий `LeadFormModal.tsx` не удаляем — остаётся для карточек авто. `LeadForm` — для страниц ноускатов и новых размещений.

`/api/lead/route.ts` — предусловие: `name` необязателен, добавлено поле `subject`.
